# FOR-SGC-08 Matriz de KPI

## 1. Objetivo
Consolidar indicadores del SGC con formula, meta, frecuencia, resultados y semaforizacion para seguimiento directivo.

## 2. Alcance
Aplica a indicadores de todos los procesos del SGC.

## 3. Referencias
- PLAN-SGC-01 Objetivos de Calidad y Seguimiento.
- PR-SGC-04 Gestion de Indicadores y Analisis de Datos.

## 4. Definiciones
- Cumplimiento (%): resultado/meta x 100.
- Tendencia: comportamiento respecto al periodo anterior.
- Semaforo: verde/amarillo/rojo segun umbral.

## 5. Responsabilidades
- Duenos de Proceso: reportar resultados y evidencias.
- Responsable de Calidad: consolidar matriz y validar datos.

## 6. Desarrollo / Metodologia
1. Definir KPI vinculado a un objetivo.
2. Registrar fuente y evidencia del dato.
3. Calcular cumplimiento y semaforo.
4. Definir accion cuando el KPI este en rojo o amarillo recurrente.

### 6.1 Umbrales sinteticos de semaforo
- Verde: cumplimiento >= 100%
- Amarillo: cumplimiento >= 95% y < 100%
- Rojo: cumplimiento < 95%
