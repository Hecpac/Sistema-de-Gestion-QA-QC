# FOR-SGC-02 Registro de No Conformidad

## 1. Objetivo
Documentar no conformidades detectadas en los procesos del SGC con clasificacion medible.

## 2. Alcance
Aplica a todas las NC detectadas en auditorias, operacion, inspecciones y reclamos.

## 3. Referencias
- PR-SGC-02 Control de No Conformidades y CAPA.

## 4. Definiciones
- NC.
- Contencion inmediata.
- Severidad de NC.

## 5. Responsabilidades
- Solicitante: registrar NC con evidencia objetiva.
- Dueno de Proceso: evaluar severidad y plan de contencion.
- Responsable de Calidad: validar clasificacion y seguimiento.

## 6. Desarrollo / Metodologia
1. Registrar datos de la NC y requisito incumplido.
2. Calificar impacto y frecuencia (escala 1-5).
3. Calcular nivel = impacto x frecuencia y clasificar.
4. Definir contencion y tiempo objetivo segun severidad.
5. Referenciar CAPA cuando aplique.

### 6.1 Criterio de severidad
| Nivel (Impacto x Frecuencia) | Clasificacion |
|---:|---|
| 1-6 | Baja |
| 7-12 | Media |
| 13-25 | Alta |
