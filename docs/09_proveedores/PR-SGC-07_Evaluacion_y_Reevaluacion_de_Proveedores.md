# PR-SGC-07 Evaluacion y Reevaluacion de Proveedores

## 1. Objetivo
Definir criterios y metodologia para seleccionar, evaluar y reevaluar proveedores.

## 2. Alcance
Aplica a proveedores que impactan la conformidad del producto o servicio.

## 3. Referencias
- ISO 9001:2015, clausula 8.4 Control de procesos, productos y servicios externos.
- FOR-SGC-12 Matriz de Homologacion y Evaluacion de Proveedores.

## 4. Definiciones
- Proveedor critico.
- Homologacion.
- Reevaluacion.

## 5. Responsabilidades
- Compras: gestionar evaluaciones.
- Dueno de Proceso: validar desempeno tecnico.
- Responsable de Calidad: asegurar trazabilidad.

## 6. Desarrollo / Metodologia
1. Definir criterios de homologacion.
2. Evaluar proveedor previo al alta.
3. Reevaluar periodicamente por desempeno.
4. Definir acciones ante incumplimientos.

## 7. Registros asociados
- REG-SGC-PROV - Evaluacion de proveedores.
