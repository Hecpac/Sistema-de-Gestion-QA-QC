# PR-SGC-06 Gestion de Riesgos y Oportunidades

## 1. Objetivo
Definir metodologia para identificar, evaluar, tratar y monitorear riesgos y oportunidades del SGC.

## 2. Alcance
Aplica a todos los procesos del SGC y a cambios relevantes del contexto.

## 3. Referencias
- ISO 9001:2015, clausula 6.1 Acciones para abordar riesgos y oportunidades.
- FOR-SGC-10 Matriz de Riesgos y Oportunidades.
- FOR-SGC-11 Plan de Tratamiento y Seguimiento de Riesgos.

## 4. Definiciones
- Riesgo.
- Oportunidad.
- Nivel de riesgo.

## 5. Responsabilidades
- Duenos de Proceso: identificar riesgos y ejecutar tratamientos.
- Responsable de Calidad: consolidar matriz y seguimiento.
- Direccion: aprobar criterios y prioridades.

## 6. Desarrollo / Metodologia
1. Identificar riesgos y oportunidades por proceso.
2. Evaluar probabilidad e impacto con criterio definido.
3. Priorizar niveles de riesgo.
4. Definir plan de tratamiento y responsables.
5. Monitorear eficacia de controles.

## 7. Registros asociados
- REG-SGC-RISK - Matriz de riesgos y oportunidades.
- REG-SGC-RISK-PLAN - Plan de tratamiento y seguimiento.
