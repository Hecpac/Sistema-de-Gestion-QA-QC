# IT-SGC-03 Instructivo de Control de Calidad Mecanica

## 1. Objetivo
Definir el metodo de verificacion mecanica para asegurar montaje, ajustes y pruebas funcionales antes de la liberacion de sistemas mecanicos.

## 2. Alcance
Aplica a montaje de equipos, alineacion, torqueado, pruebas funcionales y cierre de punch list en sistemas mecanicos de los proyectos activos.

## 3. Referencias
- PR-SGC-02 Control de No Conformidades y CAPA.
- PR-SGC-04 Gestion de Indicadores y Analisis de Datos.
- FOR-SGC-03 Plan y Registro CAPA.

## 4. Definiciones
- Punch list: listado de pendientes tecnicos detectados en inspeccion.
- Prueba funcional: ensayo que confirma operacion dentro de parametros definidos.

## 5. Responsabilidades
- Jefatura de Ingenieria Mecanica: aprobar criterios tecnicos de aceptacion.
- Supervisor Mecanico: ejecutar inspeccion y documentar resultados.
- Coordinacion de Calidad: validar cierre de hallazgos y evidencias.

## 6. Desarrollo / Metodologia
1. Verificar completitud del montaje y disponibilidad de instrumentos de medicion calibrados.
2. Ejecutar inspeccion visual, dimensional y funcional del sistema mecanico.
3. Registrar desviaciones y abrir no conformidad cuando el criterio de aceptacion no se cumpla.
4. Definir y ejecutar CAPA para cada no conformidad critica.
5. Liberar sistema solo cuando hallazgos mecanicos esten cerrados y evidenciados.

## 7. Registros asociados
- REG-SGC-NC - Registro de no conformidad asociado a desviaciones mecanicas.
- REG-SGC-CAPA - Plan y registro CAPA para cierre de causas raiz.
