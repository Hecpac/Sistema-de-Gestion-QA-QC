# Documentos externos

Aquí se referencian o almacenan (si aplica) documentos externos necesarios para operar:
- normas
- reglamentos
- requisitos de cliente
- fichas técnicas
- manuales de fabricante

Recomendación:
- Mantener un listado de control (puede ser parte de `docs/_control/lmd.yml` o un archivo separado).
- Registrar versión/edición y fuente oficial.
