# FOR-SGC-01 Solicitud de Creacion / Cambio Documental

## 1. Objetivo
Definir el formato oficial para solicitar la creacion, modificacion u obsolescencia de documentos controlados del SGC.

## 2. Alcance
Aplica a cualquier solicitud de cambio documental que afecte politicas, procedimientos, instructivos, formatos, planes o especificaciones del SGC.

## 3. Referencias
- PR-SGC-01 Control de Documentos y Registros.
- docs/_control/lmd.yml - Lista Maestra de Documentos.

## 4. Definiciones
- Solicitante: persona que propone el alta o cambio del documento.
- DP: Dueno de Proceso responsable de la validacion tecnica.
- RC: Responsable de Calidad responsable del control documental del SGC.

## 5. Responsabilidades
- Solicitante: debe completar la solicitud con informacion verificable.
- DP: debe evaluar impacto tecnico y definir acciones requeridas.
- RC: debe validar coherencia documental y controlar la ejecucion del cambio.
- Aprobador final: debe autorizar cuando aplique segun matriz de autoridad.

## 6. Desarrollo / Metodologia
1. El solicitante completa este formato con motivo y alcance del cambio.
2. El DP y el RC evalúan impacto en procesos, riesgos, capacitacion y registros.
3. Se define decision (aprobado/rechazado) y plan de implementacion.
4. Al cierre, se verifica emision de nueva version, retiro de obsoletos y comunicacion.
5. El registro llenado se conserva conforme a la matriz de registros vigente.

## 7. Registros asociados
- REG-SGC-CDC - Solicitud de Creacion/Cambio Documental (llenada).
- REG-SGC-COM - Evidencia de distribucion/comunicacion de documentos.
